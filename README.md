# Hệ thống vệ tinh quỹ đạo thấp (LEO)

## Mô tả
Dự án nghiên cứu về hệ thống vệ tinh quỹ đạo thấp (LEO) trong lĩnh vực truyền thông và mạng máy tính.

## Nội dung thực hiện
- Tìm hiểu kiến trúc hệ thống vệ tinh LEO
- Phân tích nguyên lý hoạt động của hệ thống
- Nghiên cứu mô hình truyền thông vệ tinh
- Tổng hợp các công nghệ và ứng dụng liên quan

## Nội dung chính
- Kiến trúc hệ thống vệ tinh
- Mô hình kết nối và truyền dữ liệu
- Ứng dụng thực tế của LEO (Starlink, Internet vệ tinh)
- Ưu điểm và hạn chế của hệ thống

## Tài liệu
- File Word: báo cáo chi tiết
- File PowerPoint: trình bày dự án
- Hình ảnh minh họa hệ thống

## Công cụ sử dụng
- Omnet++
- Microsoft Word
- Microsoft PowerPoint

## Kết quả
Hiểu rõ cách hoạt động của hệ thống vệ tinh LEO và ứng dụng trong mạng hiện đại.
